# python-webview-vue

This template should help get you started developing windows gui with python + vue.

### Project Init

```sh
cd frontend
npm install
```

```sh
pip install -r requirements.txt
```

### Project Start

```sh
cd frontend
npm run dev
```

```sh
python index.py
```

### Project Build

```sh
cd frontend
npm run build
```

```sh
pyinstaller build.spec
```

### Project Clean

```sh
Remove-Item -Recurse -Force dist,out
```
