# python-webview-vue

This template should help get you started developing windows gui with python + vue.

### Project Init

```sh
npm install
```

```sh
.\.venv\Scripts\pip install -r requirements.txt
```

### Project Start

```sh
npm run start-vue
```

```sh
npm run start-py
```

### Project Build

```sh
npm run build-vue
```

```sh
npm run build-py
```

### Project Clean

```sh
Remove-Item -Recurse -Force build, dist, gui
```
